# LitFlow: An Interactive Visual Analytics System for Dynamic Literature Review Exploration and Sensemaking рџ“љ

LitFlow is a full-stack web application designed to support researchers, students, and academicians in conducting dynamic literature reviews through interactive visual analytics. The platform enables users to explore, organize, analyze, and make sense of large collections of research papers efficiently.

By combining modern web technologies with intuitive visual interfaces, LitFlow simplifies the research process and helps users discover meaningful insights from academic literature.
